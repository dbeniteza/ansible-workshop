# Glossary

## Inventory

Collection of managed systems.

## Playbook

Automation definition written in YAML.

## Task

Single automation action.

## Module

Reusable automation component.

## Role

Reusable automation structure.

## Collection

Package containing automation content.

## Fact

Information gathered from a managed host.

## Execution Environment

Containerized automation runtime.

## Automation Controller

Centralized management platform for Ansible automation.

## RBAC

Role-Based Access Control.

## Workflow

Sequence of connected automation jobs.
