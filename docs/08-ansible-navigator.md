# Ansible Navigator

## What Is It?

Ansible Navigator is the modern interface for developing and running automation.

## Key Features

- Playbook execution
- Collection browsing
- Documentation lookup
- Execution environment management

## Running a Playbook

```bash
ansible-navigator run site.yml
```

## Using Standard Output

```bash
ansible-navigator run site.yml -m stdout
```

## Execution Environments

Execution Environments provide:

- Consistent dependencies
- Portable execution
- Reproducible automation
