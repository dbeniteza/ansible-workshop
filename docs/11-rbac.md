# Role-Based Access Control

## What Is RBAC?

RBAC allows administrators to control who can access automation resources.

## Main Objects

### Users

Individual accounts.

### Teams

Groups of users.

### Organizations

Logical container for resources.

## Benefits

- Security
- Delegation
- Governance
- Separation of Duties

## Example

Operations Team:

- Execute playbooks

Automation Team:

- Modify playbooks

Administrators:

- Full access
