# Automation Controller

## What Is Automation Controller?

Automation Controller provides centralized management for Ansible automation.

## Core Concepts

### Organization

Logical grouping of resources.

### Inventory

Managed endpoints.

### Credentials

Authentication information.

### Projects

Git repositories containing automation.

### Job Templates

Reusable execution definitions.

## Benefits

- Centralized execution
- Delegation
- Auditing
- Scheduling
- API access

## Typical Flow

Inventory
→ Credentials
→ Project
→ Job Template
→ Job Execution
