# Ansible Workshop Agenda

## About this Workshop

This workshop provides a practical introduction to Ansible and Red Hat Ansible Automation Platform. Participants will learn core automation concepts through guided explanations, demonstrations, and hands-on labs.

---

## Topics

- Introduction to Ansible
- How Ansible works
- Inventories
- Modules
- Playbooks

## Lab

- Exercise 1.1: Inventories and Playbooks

### Learning Outcomes

Participants will be able to:

- Create static inventories
- Execute ad hoc commands
- Build basic playbooks
- Run automation against managed hosts
