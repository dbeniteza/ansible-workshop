## Template Module

The template module:

```yaml
ansible.builtin.template
```

copies a Jinja2 file and renders all variables before writing the destination file.

Example:

```yaml
- name: Deploy template

  ansible.builtin.template:

    src: httpd.conf.j2
    dest: /tmp/httpd.conf
```

```yaml
server_admin: workshop@example.com
```

## Key Takeaways

Templates are one of the most powerful Ansible features because they allow:

- Dynamic configuration generation
- Variable substitution
- Fact-based customization
- Reusable automation

Most real-world Ansible projects rely heavily on templates for application and infrastructure configuration management.
